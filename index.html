ধাপ ১: Python ফাইল ডাউনলোড করুন

নিচে দেওয়া ডাউনলোড বাটন থেকে Python কোডটি ডাউনলোড করুন এবং আপনার কম্পিউটারে সেভ করুন (যেমন: youtube_downloader.py)

ধাপ ২: প্রয়োজনীয় লাইব্রেরি ইন্সটল করুন

Terminal বা Command Prompt খুলে নিচের কমান্ডটি রান করুন:

pip install pytube pillow requests

ধাপ ৩: প্রোগ্রামটি রান করুন

Terminal বা Command Prompt এ নিচের কমান্ডটি লিখে Enter চাপুন:

python youtube_downloader.py

ধাপ ৪: YouTube ভিডিও ডাউনলোড করুন

প্রোগ্রামটি চালু হলে:

  • YouTube ভিডিওর URL দিন এবং "Fetch Video" বাটনে ক্লিক করুন
  • ভিডিওর বিবরণ দেখার পর ডাউনলোড কোয়ালিটি সিলেক্ট করুন
  • "Download Video" বাটনে ক্লিক করুন এবং ডাউনলোড লোকেশন সিলেক্ট করুন
  • ডাউনলোড শেষ হলে ভিডিওটি আপনার সিলেক্টেড ফোল্ডারে সেভ হবে
নোট: এই টুলটি শুধুমাত্র ব্যক্তিগত ব্যবহারের জন্য। যে কোনো ভিডিও ডাউনলোড করার আগে দয়া করে কপিরাইট নিয়মগুলি পরীক্ষা করুন।

ইন্সটলেশন সমস্যা

যদি pip install করতে সমস্যা হয়, তবে নিশ্চিত করুন যে Python এবং pip সঠিকভাবে ইন্সটল করা আছে। pip আপডেট করতে নিচের কমান্ড ব্যবহার করুন:

python -m pip install --upgrade pip

প্রোগ্রাম চালাতে সমস্যা

প্রোগ্রাম চালানোর সময় যদি কোনো error দেখায়, তবে নিশ্চিত করুন যে সকল প্রয়োজনীয় লাইব্রেরি সঠিকভাবে ইন্সটল করা আছে:

pip install pytube pillow requests
